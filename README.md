# Rule-Based Chatbot using Python

## 📌 Project Overview
This project is a simple rule-based chatbot developed using Python. The chatbot responds to predefined user inputs using conditional statements (`if-else`) and runs continuously until the user chooses to exit.

The project demonstrates basic Artificial Intelligence concepts, control flow, and decision-making logic.

## 🎯 Objective
To create a chatbot that:
- Handles greetings from users
- Responds to predefined questions or commands
- Supports exit commands
- Runs in a continuous loop for conversation

## 🛠 Technologies Used
- Python 

## 🚀 Features
- Greeting responses (Hello, Hi, Hey)
- Basic conversation handling
- Exit command support
- Continuous interaction using loops
- Simple and beginner-friendly implementation

## 💻 Sample Interaction

```
ChatBot: Hello! Type 'bye' to exit.
You: Hello
ChatBot: Hello!
You: How are you?
ChatBot: I'm fine!
You: What is your name
ChatBot: I am a simple chatbot.
You: bye
ChatBot: Goodbye!
```

##  Concepts Used
- If-Else Statements
- Loops (`while`)
- User Input Handling
- Decision Making
- Basic AI Chatbot Logic
